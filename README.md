## What is this

A dark, elegant single-page site designed for a fictional law firm. Navy background, gold accents, serif headings — the kind of visual language that says "we take your case seriously" without saying a word.

No frameworks, no build tools, no dependencies. Just clean HTML, CSS and a few lines of vanilla JS for the contact form. Loads fast, works everywhere, looks sharp on any screen.

---

## Stack

HTML5 · CSS3 (Grid, Flexbox, custom properties) · Vanilla JS · Google Fonts (DM Serif Display + DM Sans) · Deployed on Netlify
