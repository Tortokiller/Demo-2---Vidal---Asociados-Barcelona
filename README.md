# ⚖️ Vidal & Asociados – Law Firm Website

This is a fictional law firm website based in Barcelona, built as a practice project using only **HTML, CSS, and vanilla JavaScript**. The goal was to create something that feels realistic, clean, and professional—like a real legal website you might actually come across online.

It’s mainly a UI/UX exercise focused on layout, design consistency, and a bit of simple interactivity.

---

## 🌐 What this project includes

This is a complete landing page for a law firm, designed to feel structured and trustworthy:

- Fully responsive layout (mobile, tablet, desktop)
- Fixed navigation with smooth scrolling
- Hero section with key stats and call-to-action buttons
- Practice areas presentation
- Team section with lawyer profiles
- Contact form with simulated interaction
- Dark, elegant design with gold accents
