# ⚖️ Vidal & Asociados – Law Firm Website

This is a fictional law firm website based in Barcelona, built as a practice project using only **HTML, CSS, and vanilla JavaScript**. The goal was to create something that feels realistic, clean, and professional—like a real legal website you might actually come across online.

It’s mainly a UI/UX exercise focused on layout, design consistency, and a bit of simple interactivity.

---

## 🌐 What this project includes

This is a complete landing page for a law firm, designed to feel structured and trustworthy:

- Fully responsive layout (mobile, tablet, desktop)
- Fixed navigation with smooth scrolling
- Hero section with key stats and call-to-action buttons
- Practice areas presentation
- Team section with lawyer profiles
- Contact form with simulated interaction
- Dark, elegant design with gold accents

## 🧠 Tech stack

Nothing fancy here—just the fundamentals, used properly:

- HTML5 for structure and semantics
- CSS3 for layout, styling, and responsiveness
- Vanilla JavaScript for simple form interaction
- Google Fonts for typography

---

## 🎨 Design approach

The visual style was inspired by real law firms: serious, minimal, and trustworthy.

- Dark navy base → professionalism and stability
- Gold accents → elegance and prestige
- Serif font for headings + sans-serif for body text

Everything is designed to feel calm, structured, and easy to read—no unnecessary distractions.

---

## 🧭 Page sections

### 🏠 Hero section
This is the first impression of the site. It introduces the firm with a clear message, a short description, and some “experience stats” like years in practice and number of cases handled.

---

### ⚖️ Practice areas
Three main legal services are showcased:

- Civil Law  
- Commercial Law  
- Labor Law  

Each one is briefly explained in a simple, easy-to-scan format.

---

### 🧩 Working method
A simple step-by-step breakdown of how the firm handles cases:

1. Analyze the situation  
2. Build a legal strategy  
3. Defend the client in court  

The idea is to make the process feel clear and transparent.

---

### 👥 Team
A small fictional team of lawyers is presented to make the site feel more real and personal.

---

### 📩 Contact form
A basic contact form where users can “send” a consultation request.

There’s no backend connected, but it still behaves like a real form:
- Shows a success message after submission
- Clears the inputs
- Automatically hides the message after a few seconds

It’s a simple touch that adds life to the page.

---

## ⚙️ JavaScript behavior

The JavaScript is intentionally minimal. It just handles the form interaction and prevents the page from reloading when submitted.

Nothing complex—just enough to simulate a real-world user experience.

---

## 📱 Responsive design

The layout adapts smoothly across devices:

- Desktop: multi-column layout with spacious sections
- Tablet: adjusted spacing and typography scaling
- Mobile: single-column layout for better readability

The goal was to keep it clean and usable everywhere, without overcomplicating things.

---

## 🚀 Possible improvements

If I keep working on this project, these would be the next steps:

- Add a mobile hamburger menu
- Connect the contact form to a real backend or email service
- Add scroll animations for a smoother feel
- Create multilingual support (English / Spanish)

## 💬 Final thoughts

This is a simple project, but it was a good exercise in building something that feels real. It’s less about complexity and more about creating a professional-looking experience using only the basics.

Sometimes, keeping things simple is what makes a design actually work.
